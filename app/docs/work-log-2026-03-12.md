# 빌드업 (Build-Up) — 작업 일지

**날짜:** 2026-03-12
**작업자:** ymjin
**목표:** 기존 Vite+React UI 프로토타입을 Next.js + Supabase 풀스택 앱으로 마이그레이션

---

## 📌 작업 배경

기존 `frontend/` 디렉토리에는 Vite + React 19 기반의 대시보드 UI 프로토타입이 있었음.
`docs/Plan.md`에 정의된 기술 스택(Next.js, Supabase, TanStack Query)대로 새 `app/` 디렉토리에 풀스택 앱을 새로 구성함.

---

## ✅ 완료된 작업

### 1. Next.js 프로젝트 초기화
- `build_up/app/` 에 Next.js 16.1.6 (App Router + TypeScript + Tailwind CSS 4) 프로젝트 생성
- 주요 의존성 설치:
  - `@supabase/supabase-js`, `@supabase/ssr` — 인증 및 DB
  - `@tanstack/react-query` — 서버 상태 관리
  - `@dnd-kit/core`, `@dnd-kit/sortable` — 칸반 드래그&드롭
  - `lucide-react` — 아이콘
  - `clsx`, `tailwind-merge` — 유틸리티

### 2. 디자인 에셋 및 CSS 테마 마이그레이션
- `frontend/public/design/` → `app/public/design/` 복사 (빌드업 아이콘 등)
- 기존 CSS 테마 변수 그대로 이전:
  - 오렌지 테마: `--color-orange-base`, `--color-orange-primary(#FB923C)`, `--color-orange-secondary(#EA580C)`
  - 블루 테마: `--color-blue-base`, `--color-blue-primary(#3B82F6)`
  - 글래스모피즘 유틸리티 클래스 `.glass`
- 폰트: `Outfit`(헤딩) + `Inter`(본문) — Google Fonts

### 3. 타입 시스템 정의 (`types/index.ts`)
| 타입 | 설명 |
|------|------|
| `Profile` | 사용자 프로필 |
| `Project` | 프로젝트 (상태, 진행률 포함) |
| `Task` | 태스크 (칸반 카드) |
| `ProjectMember` | 프로젝트 멤버 + 역할 |
| `Comment` | 피드백 코멘트 |
| `Activity` | 활동 로그 |

### 4. Supabase 설정
- `lib/supabase/client.ts` — 브라우저 클라이언트
- `lib/supabase/server.ts` — 서버 컴포넌트용 클라이언트
- `lib/supabase/middleware.ts` — 인증 세션 갱신 + 라우트 보호
- `proxy.ts` — Next.js 16 proxy 진입점 (구 middleware.ts)
- `supabase-schema.sql` — 전체 DB 스키마 (RLS 정책 포함)

### 5. DB 스키마 설계
| 테이블 | 설명 |
|--------|------|
| `profiles` | auth.users 연동, 자동 생성 트리거 |
| `projects` | 프로젝트 메타데이터 |
| `tasks` | 태스크 (상태: todo/in_progress/done) |
| `project_members` | 멤버-역할 매핑 |
| `comments` | 피드백 코멘트 |
| `activities` | 활동 로그 |

### 6. TanStack Query 훅 구현 (`lib/queries.ts`)
- `useProjects`, `useProject`, `useCreateProject`, `useUpdateProject`, `useDeleteProject`
- `useTasks`, `useCreateTask`, `useUpdateTask`, `useDeleteTask`
- `useMembers`, `useInviteMember`, `useRemoveMember`
- `useComments`, `useCreateComment`
- `useActivities`

### 7. 페이지 및 컴포넌트 구현

| 파일 | 설명 |
|------|------|
| `app/(app)/layout.tsx` | 사이드바 포함 앱 레이아웃 |
| `app/(app)/dashboard/page.tsx` | 대시보드 (글로벌 점수, 히트맵, 프로젝트 그리드) |
| `app/(app)/projects/page.tsx` | 프로젝트 목록 + 필터 |
| `app/(app)/projects/new/page.tsx` | 새 프로젝트 생성 폼 |
| `app/(app)/projects/[id]/page.tsx` | 프로젝트 상세 (칸반 + 코멘트 + 활동로그) |
| `app/(app)/projects/[id]/members/page.tsx` | 팀원 관리 |
| `app/auth/login/page.tsx` | 로그인 |
| `app/auth/signup/page.tsx` | 회원가입 |
| `app/auth/callback/route.ts` | OAuth 콜백 처리 |
| `app/setup/page.tsx` | Supabase 미설정 시 안내 페이지 |
| `components/layout/Sidebar.tsx` | 사이드바 내비게이션 |
| `components/kanban/KanbanBoard.tsx` | dnd-kit 기반 칸반 보드 |
| `components/kanban/KanbanCard.tsx` | 드래그 가능한 태스크 카드 |
| `components/providers.tsx` | TanStack Query 프로바이더 |

### 8. 개발 환경 설정
- `.claude/launch.json` — 두 dev 서버 설정 등록
  - Next.js App (port 3000)
  - Vite Frontend 프로토타입 (port 5173)
